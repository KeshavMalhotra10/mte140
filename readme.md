# MTE 140 — Data Structures and Algorithms

This repository contains implementations of fundamental data structures and algorithms developed as part of MTE 140. The focus is on understanding core concepts through manual implementation in C++.

---

## Contents

### Linked Lists
- Singly linked list implementation
- Doubly linked list implementation
- Insertion and deletion operations
- Traversal and search

### Queues
- Array-based and linked list implementations
- Standard FIFO operations
- Basic applications (e.g., breadth-first traversal)

### Binary Search Trees
- Node insertion and deletion
- Tree traversal (inorder, preorder, postorder)
- Search operations

### Recursion
- Classic recursive problems
- Base and recursive case design
- Applications to data structures and problem solving

### Sorting
- Bubble sort
- Insertion sort

---

## Objectives
- Develop a strong foundation in core data structures
- Improve algorithmic thinking and problem-solving skills
- Write clear, efficient, and maintainable C++ code

---

## Notes
All implementations are written from scratch for learning purposes, without reliance on STL equivalents unless explicitly noted.
